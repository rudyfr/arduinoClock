Detailed Description

A Clock Class for Arduino.

The Clock is based on: http://www.faludi.com/examples/open-source-arduino-clock, but this version is object-based. Converted to a libtrary by following the tutorial at: http://arduino.cc/en/Hacking/LibraryTutorial. Finally, the code comments are in Doxygen style.

Construct a Clock object.

boolean Clock::run ( )

Runs the Clock.

If the Clock is at least 1 second behind, 1 second is added to the Clock object. If the Clock is not run at least once a second, it will fall further behind.
